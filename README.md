# ChronHR — Marketing Website

Public marketing site for [ChronHR](https://chronhr.com), an AML intelligence platform built for compliance analysts.

## Stack

- React
- IBM Plex Sans
- CSS Modules

## Getting Started

```bash
npm install
npm run dev
```

## Pages

- `/` — Landing
- `/about` — About
- `/team` — Meet the team
- `/pricing` — Pricing & ROI calculator
- `/get-started` — Contact
